Grundsatz: Datenminimierung

Diese Website ist bewusst so gebaut, dass beim bloßen Aufruf so wenige personenbezogene Daten wie möglich verarbeitet werden:

  • Keine Cookies
  • Kein Tracking, keine Analyse-Tools (z.B. Google Analytics, Matomo, Plausible)
  • Keine Social-Media-Plugins, keine eingebetteten Videos
  • Keine Kontaktformulare, keine Kommentare, keine Newsletter
  • Keine Werbung, keine Werbe-Tracker

Hosting (GitHub Pages)

Diese Website wird über GitHub Pages ausgeliefert. Anbieter ist die GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA („GitHub").

Beim Aufruf der Seite erhebt GitHub aus technischen Gründen Server-Log-Daten. Dazu gehören üblicherweise:

  • Gekürzte IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene URL
  • Verwendeter Browser und Betriebssystem (User-Agent)
  • Referer-URL (die Seite, von der Sie kamen, sofern übertragen)

Diese Daten sind erforderlich, um die Seite sicher und stabil ausliefern zu können (Abwehr von Missbrauch, Lastverteilung, Fehleranalyse). Eine Verknüpfung dieser Daten mit Ihrer Person findet durch mich nicht statt.

Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse am sicheren Betrieb des Angebots).

Die Datenübermittlung in die USA erfolgt auf Grundlage des EU-US Data Privacy Framework. Details zur Datenverarbeitung durch GitHub finden Sie hier: GitHub Privacy Statement.

Schriften (lokal gehostet)

Diese Website verwendet die Schriftart „Open Sans" (SIL Open Font License). Die Schriftdateien werden direkt vom Server dieser Website ausgeliefert. Es findet keine Verbindung zu Google Fonts oder anderen Drittanbietern statt.

Externe Links (YouTube)

Auf dieser Seite befinden sich Links zu Predigt-Videos auf YouTube (betrieben von der Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland).

Beim bloßen Anzeigen dieser Seite werden keine Daten an YouTube oder Google übermittelt — die Links sind reine Textverweise, es werden keine Vorschau- oder Iframe-Einbettungen geladen.

Erst wenn Sie aktiv auf den Button „Zur Predigt" klicken, öffnet sich ein neues Browser-Fenster mit YouTube. Ab diesem Zeitpunkt gelten ausschließlich die Datenschutzbestimmungen von Google: policies.google.com/privacy.

Speicherung im Browser (Local Storage)

Diese Website speichert nichts im Local Storage und keine Cookies. Ihre Einstellungen (z.B. ob Sie das Rad oder die Liste sehen möchten) bleiben nur für die aktuelle Sitzung erhalten und werden nach dem Schließen des Tabs vergessen.
